Using margin properties individual:

This div element has a top margin of 100px, a right margin of 150px; a bottom margin of 100px, and a left margin of 80px.

Using margin property with 4 values:

margin 25px 50px 75px 100px

Using margin property with 3 values:

margin: 25px 50px 75px
  • margin top: 25px
  • margin right and left: 50px
  • margin bottom 75px

Using margin property with 2 values:

margin 25px 50px
  • top and bottom margin: 25px
  • right and left margin: 50px

Using margin property with one value:

margin: 25px;
  • all four margins are 25px

this paragraph inherit left margin from the parent div element.

Auto margin.

This div will be horizontally center because it has margin auto;
